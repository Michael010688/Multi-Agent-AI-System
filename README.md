# Multi-Agent AI System

This project implements a simple multi-agent AI system with three agents:

- Planner Agent
- Research Agent
- Writer Agent

## Features

- Task planning
- Information collection
- Final response generation

## Tech Stack

- Python
- LangChain
- OpenAI API

## Architecture

User Task
   ↓
Planner Agent
   ↓
Research Agent
   ↓
Writer Agent
   ↓
Final Output
